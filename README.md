# NYT Article Clone

A [New York Times Article](https://www.newscientist.com/article/2286218-ancient-lake-in-marss-gale-crater-may-have-actually-been-a-small-pond/) clone created through [TheOdinProject](https://www.theodinproject.com) online course. The main objective of this project is to replicate a New York Times article, focusing on HTML/CSS and ignoring all JavaScript functionality. This project is for learning purposes only.
